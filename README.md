# ☕ Coffee Shop Website

A simple and responsive Coffee Shop website built using HTML and CSS. This project is designed to showcase a modern UI for an online coffee store with attractive product sections and a clean layout.

## 📌 Project Overview
This website allows users to explore different coffee types and milkshakes with a visually appealing design. It is a front-end project focused on improving UI/UX skills and web development basics.

## ✨ Features
- Modern and clean user interface  
- Coffee and milkshake categories  
- Product cards with images and pricing  
- Buttons like “Buy Now” and “Shop Now”  
- Simple and structured layout  

## 🛠️ Technologies Used
- HTML5  
- CSS3
